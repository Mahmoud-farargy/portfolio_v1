Resources
Resources that have helped me throughout my self-teaching process
Udemy courses:

These courses were highly effective and advantageous, they have played a huge part in helping me reach the level of knowledge where I'm at now as they provided a full in-depth curriculum of each language and framework with an excellent explanation of each lesson in a way that is understandable and easy to follow.
I've bought a variety of Udemy courses from different instructors, watched them, understood each point presented and applied what I've learned in my text editor.

FreeCodeCamp website:

I've actually spent countless hours learning on this website alone. In this period, I managed to completely finish the client-side section and the challenges that are provided there. The hardest part was the intermediate Javascript algorithms but I absolutely enjoyed every second of that time and got familiar with solving problems and knew that in order to unravel the most complex algorithm you must divide it into little chucks then solve each part individually and eventually create the entire picture which is the web application that's intended to be done. My timeline on freecodecamp

Youtube Tutorials:

Before learning a language or framework I like to check out crash courses about it on YouTube to understand its basic concepts then gradually pave my way for learning more advanced concepts and dive deeper in it from there.
Also, I used to refer to online tutorials whenever I get stuck in making a certain effect or feature then I make sure to get a full grasp of it so I can apply it without problems next time.

Online Articles & PDFs:

In my spare time, I like to read articles about the web development industry, how the internet works and how to implement some features into a web page or make an element appear in a certain way.
